Para a execução do programa, recomenda-se iniciar um cmd que servirá para executar o código
do servidor e um para o código do cliente. Para executar eles, basta ir até a pasta "SegComp_TP2"
em cada um dos prompts, e em um executar o comando "python server.py", e no outro "python client.py",
nessa ordem, considerando o sistema operacional Windows. Para usuários de outros sistemas operacionais,
realizar os passos análogos.

O código do cliente utiliza os módulos:
-http.client
-ssl

O código do servidor utiliza os módulos:
-http.server
-ssl
-BytesIO
